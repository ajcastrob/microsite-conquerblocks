# ConquerBlocks Microsite

Proyecto educativo de frontend web — HTML semántico, Sass BEM y JavaScript vanilla.

## Stack

Vite · Sass · Vanilla JS · Web Components · Remix Icons

## Desarrollo

```bash
pnpm install
pnpm dev     # localhost:1234
pnpm build   # → dist/
```
